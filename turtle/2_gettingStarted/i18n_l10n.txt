Important variables associated with internationalization and localization

LANG       default for any LC_* variable not otherwise set
LC_ALL     value overriding all other LC_* variables
LC_COLLATE locale name for collating (sorting)
LC_CTYPE                   character types (alphabetic, puncutuation, digit, &c)
LC_MESSAGES                affirmative/negative responses/messages (POSIX)
LC_MONETARY                currency formatting
LC_NUMERIC                 number formatting
LC_TIME                    time formatting


# List all locales:
> locale -a

# Query details:
> LC_ALL=da locale -ck LC_TIME # get locale info for Danish time

# Get help
> LC_ALL=it_IT ls --help # get help for GNU ls in Italian
